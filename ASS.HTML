2.DIFFERENCIATE BETWEEN Var/Let/Const.

When to Use Let and Const.

Var can be tricky because its scope is either global or within a function, which can lead to bugs. To avoid these issues:

*Use Let when you know a variable's value might change later in your code.

*Use Const for variables that should never change once you set them.

Using LET and CONST makes your code easier to understand and helps prevent errors caused by unexpected variable changes.

VAR LET CONST
The scope of a Var variable is functional or global scope It can be updated and re-declared in the same scope. It can be declared without initialization.
The scope of a Let variable is block scope. It can be updated but cannot be declared in the same scope It can be declared without Initialization.
The scope of a Const Variable is block scope It can neither be updated or re-declared in any scope. It cannot be declared without initialization.
